Este es el primer cuadro en html

HTML significa lenguaje de marcado de hiper texto utilizado para el
desarrollo de paginas web. Para programar con este lenguaje el usuario
debe disponer de un editor de codigo como pueden ser Visual Estudio Code,
Note Pad, Block de Notas, donde pueda desarrollar el codigo de la pagina
que desarrolle.

La etiqueta html es la primera y en ella se crean todas las demas etiquetas
se puede decir que es la caja principal la cual contiene las demas cajas o
o etiquetas. La etiqueta head es donde se coloca el encabezado principal de la pagina web que
creamos, en html5 se declaran dentro de ella la etiqueta meta, title y la de estilos
o stylesheet.

La etiqueta meta en esta se definen el juego de caracteres a utilizar para mostrar el
documento en pantalla.
